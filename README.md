## Install

Run `yarn add @cognite/gearbox` or `npm install @cognite/gearbox --save`.

Install additional dependencies:
1.  Using Yarn `yarn add @cognite/sdk @cognite/react-auth antd styled-components`
2.  Using NPM `npm i @cognite/sdk @cognite/react-auth antd styled-components --save`
